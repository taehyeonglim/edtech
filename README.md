# 교육방법 및 교육공학

교육방법 및 교육공학 10강 HTML 교안과 PDF 배포본을 모아 둔 GitHub Pages용 정적 사이트입니다.

## 구성

- `index.html`: 10강 전체 교안 링크
- `chapters/chapter-*/slides/deck.html`: 강의용 HTML 슬라이드
- `chapters/chapter-*/slides/deck.pdf`: PDF 배포본
- `chapters/chapter-*/images/`: 슬라이드 이미지 자산

GitHub Pages는 `main` 브랜치 루트에서 서비스하도록 설정합니다.
